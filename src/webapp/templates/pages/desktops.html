How to connect to desktops

  • Spice client app

    • Windows 64bits: Install viewer - Install usb plug
    • Windows 32bits: Install viewer - - Install usb plug
    • Debian/Ubuntu based distros: sudo apt install virt-viewer -y
    • RedHat/CentOS/Fedora based distros: sudo dnf install remote-viewer -y
    • Android: Install aSpice viewer
    • MacOS: There is no spice client app currently. Use default VNC client option.
    • iOS: Install FlexVDI viewer

  • Spice & VNC browser client

    Use your browser to connect without installing apps. You should allow pop-up windows.
    • Chrome browser: Should work
    • Firefox browser: If you use self signed certificates to encrypt connection it will open a second pop-up window to accept certificate before connecting. You can close that window after accepting certificate.

  • VNC client app

    • RealPlayer Install viewer . When opening file in RealPlayer it will ask for password. You can copy&paste password from desktop details.
    • Linux: vinagre -F console.vnc
    • Android: Not tested.
    • MacOS: Open downloaded console.vncloc file with default VNC client viewer app.
    • iOS: Not tested
