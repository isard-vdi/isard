Edit quota (users)

The quota will be applied to individual users in this category/group.
Propagate will overwrite existing quotas on child items.
It is applied to a user in this order:
  • User quota, if set, else
  • User group quota, if set, else
  • User category quota, if set.

Edit limits (category/group)

The limits will be applied as the absolute maximum for this category/group.
The limits for user category/group set have priority over user quota.
Propagate will overwrite existing limits on child items.
It is applied to a user in this order:
  • User group limits, if set, else
  • User category limits, if set
