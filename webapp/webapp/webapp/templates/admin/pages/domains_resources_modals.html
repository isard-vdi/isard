For advanced network labs running inside a desktop (GNS3, EVE-NG, Packet Tracer in real mode, bridged nested VMs). Available only on OpenVSwitch and Personal interfaces outside VLAN 4095. Each option takes effect on the next desktop start; already-running desktops keep the flow set installed at their start time.

Reduced security on this network. Frames with arbitrary non-IsardVDI source MACs are accepted on this interface. IsardVDI desktop MACs (52:54:00) cannot be impersonated, but within this same VLAN a user can still spoof non-IsardVDI MACs and sniff traffic. Use only on dedicated lab networks where this trade-off is explicitly accepted.
