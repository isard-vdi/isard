# isardvdi-scheduler-client

Generated Python client for the IsardVDI `isard-scheduler` HTTP API.

## Layout

- `isardvdi_scheduler_client/` — generated by `openapi-python-client` from
  `pkg/oas/scheduler/scheduler.json` during `bash build.sh`. **Do not edit.**
- `isardvdi_scheduler_client_auth/` — hand-written helpers:
  - `build_client(service)` → `AuthenticatedClient` with a fresh service
    JWT and the correct base URL
  - `SchedulerApiError` / `raise_for_status` re-exported from
    `isardvdi_apiv4_client_auth` so every call site has a typed error.

## Usage

```python
from isardvdi_scheduler_client.api.jobs import post_scheduler_add
from isardvdi_scheduler_client.models import SchedulerAddJobRequest
from isardvdi_scheduler_client_auth import build_client, raise_for_status

with build_client("isard-apiv4") as client:
    resp = post_scheduler_add.sync_detailed(
        client=client,
        body=SchedulerAddJobRequest(...),
    )
    raise_for_status(resp)
```
